# rotation (frontend)

Static frontend (one `index.html`) that talks to the backend.

## Local usage
- Put this folder at: `localhost/public/rotation/`
- Open: `http://localhost:8888/rotation/index.html` (or your local server)
- Set API Base URL at the top to: `http://localhost:5050`

## Netlify usage
- Put this folder at: `public/rotation/` in your site.
- It will be available at: `/rotation/index.html`
- Set API Base URL to your Render service URL.
